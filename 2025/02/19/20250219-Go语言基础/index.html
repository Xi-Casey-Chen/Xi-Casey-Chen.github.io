多文件结构

Go 中同一个文件夹下的包名应该保持一致（建议与文件夹名一致），且包中的函数不能出现重名，这就使 package main 中仅存在有一个 main() 函数作为执行入口。在项目中，应将 package main 的 Go 程序放在项目的根目录。
