Package

import 导入包名，Go 在引入包时首先会在 GOPATH/src 和 GOROOT/src 两个路径中查找。因此，建议 Go 的开发尽量放在这两个路径下以减少不必要的麻烦。

Go 一个文件夹对应到一个包，包名是文件夹的路径名。同一个文件夹下的包名应该保持一致（建议与文件夹名一致），且包中的函数、变量等不能出现重名。

有时会遇到同名包的情况，这个时候需要使用到包的别名对同名包进行区分。

输出

fmt.Printf() 支持格式化输出，使用方式类比 C 的 printf()。
fmt.Sprintf() 支持格式化输出，使用方式类比于 C 的 sprintf。
Go 中额外支持的格式有：

  • %v ，按本来值输出，会自动转换，支持输出结构体和数组等
  • %+v ，在 %v 的基础上会展开结构体的字段名
  • %#v ，在 %+v 的基础上再输出结构体名（类型）
  • %T ，输出类型

fmt.Print() 无需格式化输出，其会默认将每个参数以 %v 输出。
fmt.Println() 在 fmt.Print() 基础上将内容输出并以空格分隔，默认在行尾加上换行。

数据类型

Go 中的数据类型主要有三种布尔型 bool ，整型 int 以及浮点型 float32 / float64。

整型除了有一般的 int 外，还有基于架构的变量 int8 ， int16 ， int32 ， int64 以及 uint8 ， uint16 ， uint32 ， uint64 。

浮点型有 float32 ， float64 以及复数 complex64 （32 位实数和虚数）， complex128 （64 位实数和虚数）。
